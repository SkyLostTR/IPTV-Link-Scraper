# IPTV-Link-Scraper
IPTV-Link-Scraper scrapes IPTV links from https://streamtest.in/ and saves them
as an ``m3u`` playlist.

with:
```

```

To use the command line interface directly:
```
python3 iptv_scraper.py CHANNEL_NAME --pages 2
```
